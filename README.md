# Portfolio Website Template

A clean, responsive portfolio website template built with HTML and CSS.

## Features

- Responsive design
- Modern and clean UI
- Smooth scrolling navigation
- Sections for About, Skills, Projects, and Contact
- Easy to customize

## Getting Started

1. Clone this repository
2. Customize the content in `index.html`
3. Modify the styles in `css/styles.css` to match your preferences
4. Add your own projects and information
5. Deploy to your preferred hosting platform

## Customization

- Update the text in `index.html` with your personal information
- Modify the color scheme in `css/styles.css`
- Add your own projects to the Projects section
- Update the contact information
- Add your own logo or branding
